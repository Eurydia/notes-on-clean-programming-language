# Boolean type

For built-in functions and operations on Booleans, see [[stdbool]].

## Type annotation

Booleans are annotated with `Bool`.

```
// CLEAN

expr :: Bool
expr =  1 == 1
```

## Boolean Literals

There are two Boolean literals.

```
// CLEAN

boolT :: Bool
boolT =  True

boolF :: Bool
boolF =  False
```
